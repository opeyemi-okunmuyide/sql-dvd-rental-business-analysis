# DVD Rental Business Analysis Using SQL
This project analyzes a DVD rental business database to answer business questions related to customer behaviour, inventory management, store performance, and revenue trends using the Maven Movies sample database.

The goal of this analysis was to extract meaningful insights from relational data and demonstrate how SQL can be used to support business decision-making.

## Tools Used
- SQL
- MYSQL Workbench

## Business Analysis Areas

### 1. Business and Store Analysis
- Evaluated store performance and operational metrics
- Analyzed rental activity across different locations

### 2. Inventory Analysis
- Examined inventory availability and film diversity
- Analyzed film categories and replacement costs

### 3. Customer and Revenue Analysis
- Identified customer rental patterns
- Analyzed revenue contribution and customer behaviour

### 4. Stakeholder Analysis
- Created queries designed to answer common business reporting questions

### 5. Market Analysis
- Explored trends within the rental database to identify business opportunities

## Skills Demonstrated
- Complex SQL queries
- Data aggregation
- GROUP BY and filtering
- Joins (INNER JOIN, LEFT JOIN)
- Data analysis and reporting
- Inventory analysis
- Customer segmentation
- Business problem solving

## Key Insights
- Identified the most popular film categories based on rental activity
- Determined customer groups contributing the most revenue
- Evaluated inventory diversity across stores
